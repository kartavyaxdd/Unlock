# Privacy Policy

MentalismApp is an entertainment tool intended for live performance by magicians and mentalists.

## Data practices

- No account creation
- No analytics
- No advertising identifiers
- No cloud sync
- No server-side processing
- No network transmission of spectator secrets

## Secrets entered during a routine

If a spectator voluntarily enters a code into the app's keypad, that value is used only to perform the active routine.

- The code is kept in memory for the active session
- The code is cleared when the performer resets the routine
- The code is cleared when the app starts a new routine
- The app is designed not to persist spectator secrets to storage

## Permissions

MentalismApp v1 is designed to avoid privacy-sensitive permissions such as camera, microphone, contacts, location, NFC, and Bluetooth device discovery.

Audio reveal uses standard system playback behavior.

## Intended use

MentalismApp does not read device passcodes, intercept system input, access other apps, or unlock real devices. It is a performance illusion tool only.
